VMware15 下安装 CentOS 7 的简易指南

说明

本教程意在简单快速安装 CentOS 7 ，并没有细说背后的原理。所使用的环境是：

  • VMware 15
  • Windows10

创建新的虚拟机

  1. 选择 文件 > 新建虚拟机

  1. 选择 典型，然后点击下一步

  2. 选择 稍后安装操作系统，然后下一步

  3. 客户机操作系统 选择 Linux ，版本 选择 CentOS 7 64 位，然后点击下一步

  4. 输入 虚拟机名称 和 选择要放在哪里，然后点击下一步

  5. 设置虚拟机的磁盘大小，其它的保持默认即可，无太大影响，然后点击下一步

  6. 选择 自定义硬件… ，可以根据自己的需要设置一下 cpu 和 内存，我这里设置的是 双核 2g。网络适配器保持 NAT 模式即可。CD/DVD 驱动器 中的 连接 选择 使用 ISO 映像文件，然后选择系统的镜像文件，等安装成功后可以选回 使用物理驱动器。设置好后直接关闭即可，最后点击完成。

安装系统

  1. 启动虚拟机，注意白色字体的才是当前选中项，用方向键选择最上面的那个 Install CentOS 7

  2. 建议直接选择英文，然后点 Continue

  3. 这里需要手动设置一下时间，在地图上点击中国的位置，因为现在还没法联网，可以自己到网上搜索 北京时间 然后设置进去，确认时间无误后，点击 Done

  4. 往下看到 INSTALLATION DESTINATION 这里，需要点进去一下，什么也不用动，保持默认，然后点击 Done

  5. 最后点一下 Begin Installation，然后就进入 安装环节，默认是最小安装，是没有图形界面的

  6. 安装过程中可以设置一下 root 用户的密码，点击 ROOT PASSWORD

  7. 输入两遍之后点击 Done。注意：如果使用的是数字小键盘，就需要检查一下 num lock 有没有开，否则会打不出数字来，之后安装完成进入系统的时候，也要先检查一下，要不然死活登录不上 root 账号

  8. 进度条跑完之后点击 Reboot 重启即可

  9. 重启后进入系统，输入 root 用户名和密码即可完成登录。注意检查数字小键盘问题，因为密码输入的时候是不会回显的，可以先在用户名输入的时候检查一下是否能正常输入数字

设置网络

因为平常使用 Linux 的时候不习惯直接进入，而是使用 ssh 客户端远程连接使用，这时把系统设置成 静态 ip 就很有必要了

  1. 选择 编辑 > 虚拟网络编辑器

  2. 点击 更改设置 ，此时需要管理员权限

  3. 找到并选中类型为 NAT 模式 的网卡，并且记一下网卡的名称，一会儿设置会用到

  4. 取消勾选 使用本地 DHCP 服务将 IP 地址分配给虚拟机，并且记一下下面的 子网IP 和 子网掩码 ，一会儿设置会用到

  5. 点击进入 NAT 设置，将网关最后一位的 1 改为 2

  6. 设置完成后点 应用 ，然后打开 控制面板 ，找到 网络和 Internet > 网络和共享中心 ，点击旁边的 更改适配器设置

  7. 找到刚刚设置的那张网卡，右击选择 属性

  8. 找到 Internet 协议版本 4 (TCP/IPv4) 这一项，双击进入

  9. 选择 使用下面的 IP 地址 和 使用下面的 DNS 服务器地址 。然后将刚刚记下的 子网验码 和 网关 IP 填上去。IP 地址 就填该子网下的 1 号机，即 xxx.xxx.xxx.1。DNS 服务器地址就上网搜索一些可用的填上去，我这里用的是 114 和 阿里 的。填完后点 确定 退出

  10. 上面的设置完后就启动进入虚拟机，然后使用 vi 编辑 /etc/sysconfig/network-scripts/ifcfg-ens33 文件，注意：这里有可能不是 ifcfg-ens33 文件，有些电脑上的网卡不叫 ens33 这个名字的，所以根据自己电脑的情况，找到对应的 ifcfg-xxx 文件即可。然后修改 BOOTPROTO 为 static，修改 ONBOOT 为 yes。意思是 IP 改为静态分配，并且网卡开机自启动
